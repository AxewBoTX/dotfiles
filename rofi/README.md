# Rofi Config

[Rofi](https://github.com/davatorium/rofi) is a window switcher, application launcher and dmenu replacement for Linux systems.
<br>
It a pretty great tool for running apps and it can be easily configured the way you want it to.
<br>
It a highly customisable tool both appearance and functionality vise.

You can install rofi using the following command based on your OS:-

- Debian Based Systems: `sudo apt install rofi`
- Arch Based Systems: `sudo pacman -S rofi`

In my I3WM configuration I have setup `MOD(Windows Key)+Space` to opening rofi.
