# Zsh Config

Zsh is a mordern shell that includes many features which makes it super easy and user friendly to work with the Terminal.
<br></br>
You can install zsh on your system by following the [Install Guide](https://github.com/AxewBoTX/dotfiles/blob/main/Install%20Guide.md) in this repo. 
<br>
Zsh can be integrated with a variaty of plugins that are super helpfull. I use the following plugin in my zsh config:-
- [powerline10k](https://github.com/romkatv/powerlevel10k) - Its a super cool and aesthetic powerline theme for zsh.
- [zsh-auto-suggestions](https://github.com/zsh-users/zsh-autosuggestions) - This plugin adds very helpfull and user friendly auto completion for your zsh commands. It can be very helping if you easily forget the commands.
- web-search - It is a basic plugin that helps you search on google without exiting your Terminal.  Because why would you wanna exit your terminal. ^_^
